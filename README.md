# Background Removal and Color Applier

This is a basic tool for removing background and applying color to images. I built this to be able to do it quickly for company's logos that I work with. 

## Getting Started

To get started is very simple. 

1. Follow the installation guide below 
3. Create an original_files directory (must be exactly original_files)
4. Drop all the files you want inside original_files 
5. Execute python app.py. 
6. The output should be written where the console points to. 

### Prerequisites

Software needed:
* Python 3
* A package manager (pip preferred)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/rbverissimo/bg-removal-color-apply.git
   ```
2. Run and activate venv: 
   ```bash
   python -m venv venv
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
